# Wedding Invitation UI Kit

Digital wedding invitation for Elisa & Christan.

## Screens

| Screen | File | Description |
|--------|------|-------------|
| Invitation | `Invitation.jsx` | Hero screen — names, headline, date, location, CTAs |
| RSVP | `RSVP.jsx` | RSVP form with attendance toggle and success state |
| Schedule | `Schedule.jsx` | Timeline/program of the wedding day |

## Design Notes

- All text centered; generous vertical spacing
- Font stack: Great Vibes (display) · Pinyon Script (names) · Cormorant Garamond (body)
- Color: Gold `#B8922B` · Slate `#3F5059` · Cream `#FDFAF6`
- Protea illustration used as ghost watermark at 10–15% opacity
- Buttons: filled slate (primary) · outlined taupe (secondary)
- Form fields: borderless except bottom hairline, light weight serif
- No shadows on elements within the card; one ambient shadow on the card itself

## Usage

Open `index.html` to see the interactive prototype. Navigate between screens using the RSVP and Programm buttons.
